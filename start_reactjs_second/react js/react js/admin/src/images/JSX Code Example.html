4.1 Modify App.js to Add Routing

Open App.js and import the necessary components from react-router-dom.

jsx
// src/App.js
import React from 'react';
import { BrowserRouter as Router, Route, Switch, Link } from 'react-router-dom';
import HomePage from './HomePage';
import AboutPage from './AboutPage';
import ContactPage from './ContactPage';
import './App.css';

const App = () => {
  return (
    <Router>
      <div className="app">
        {/* Navigation Links */}
        <nav>
          <ul>
            <li>
              <Link to="/">Home</Link>
            </li>
            <li>
              <Link to="/about">About</Link>
            </li>
            <li>
              <Link to="/contact">Contact</Link>
            </li>
          </ul>
        </nav>

        {/* Routes */}
        <Switch>
          <Route exact path="/" component={HomePage} />
          <Route path="/about" component={AboutPage} />
          <Route path="/contact" component={ContactPage} />
        </Switch>
      </div>
    </Router>
  );
};

export default App;

Explanation:

  1. BrowserRouter as Router: This component wraps your entire app and allows you to manage routing with the HTML5 history API.
  2. Link: The Link component from react-router-dom is used to create navigation links between pages. It prevents full page reloads and only updates the URL, making it a SPA (Single Page Application).
  3. Route: The Route component is used to define the path and the component that should be displayed when the user visits that path.
  4. Switch: The Switch component ensures that only one Route is rendered at a time, even if multiple routes match the URL.
